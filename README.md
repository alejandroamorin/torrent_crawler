# Torrent Crawler
Project with several crawlers for different torrent pages

# Version
v0.2

# Dependencies:
- Switch
- Mechanize

# How to run torrent_crawler
```
perl torrent_crawler.pl "several keywords"
```
```
perl torrent_crawler.pl 'several keywords'
```
```
perl torrent_crawler.pl 'keyword'
```
